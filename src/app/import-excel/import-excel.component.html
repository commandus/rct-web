Загрузка из книги Excel

1. Выберите тип компонентов. Для компонентов с номиналом тип будет определен по единице измерения, указанной в номинале. Если оставить это поле пустым, загрузятся только строки с указанным номиналом.
2. Выберите коробку, в которую будут импортирован файл Excel. Оставьте пустым, если коробки будут добавлены в корень. Последний номер коробки указан в столбце 'A' таблицы.
3. Отметьте галочку, нужно ли добавлять номер коробки из имени файла.
Номера в имени файла
4. Выберите файл книги Excel. Все электронные таблицы (закладки) будут загружены, если формат строки окажется подходящим.
